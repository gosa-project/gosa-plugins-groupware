Festplatten

Zu jeder Partitionstabelle gehören eine oder mehrere Festplatten (wobei GOsa es durchaus erlaubt, leere Partitionstabellen zu erstellen). Diese enthalten die tatsächlich später anzulegenden Partitionen. Um eine Festplatte hinzuzufügen, werden zum einen die allgemeinen Angaben zur Festplatte sowie die einzelnen Partitionseinträge benötigt.
