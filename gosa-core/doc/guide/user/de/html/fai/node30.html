FAI-Variable bearbeiten

Um eine bestehende Variable zu bearbeiten, markieren Sie diese in der Liste benutzter Variablen und drücken Sie auf den Knopf Bearbeiten. Bearbeiten Sie die Variable und speichern Sie die Änderungen, indem Sie auf den Knopf Anwenden drücken. Wenn Sie die Änderungen verwerfen möchten, drücken Sie auf den Knopf Abbrechen.
