Importieren

Importiere LDIF Datei Drücken Sie auf den Knopf neben dem Textfeld, um die Dateiauswahl zu starten. Wählen Sie die LDIF-Datei, die Sie importieren möchten.

  • Überschreibe vorhandene Attribute: Attribute, die bereits vorhanden sind, werden mit dem Wert aus der LDIF-Datei überschrieben
  • Existierenden Eintrag überschreiben: Einträge, die bereits vorhanden sind, werden mit dem Eintrag aus der LDIF-Datei überschrieben

    Drücken Sie auf den Knopf Importieren, um den Importvorgang durchzuführen.
