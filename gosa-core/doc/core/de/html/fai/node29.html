FAI-Variable hinzufügen

Um eine Variable zu einer Variablengruppe hinzuzufügen, drücken Sie auf den Knopf Hinzufügen unterhalb der Liste benutzter Variablen. Geben Sie den zumindest Namen der Variablen, sowie den Inhalt ein (optional können Sie in das Feld Beschreibung eine kurze Beschreibung der Variablen eingeben) und drücken Sie auf den Kopf Anwenden, um die neue Variable zu speichern.
