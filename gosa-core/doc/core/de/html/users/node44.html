Bearbeiten:

Wenn Sie den Knopf Bearbeiten gedrückt haben, sehen Sie eine zweigeteilte Seite. Auf der linken Seite sehen Sie die aktuell gültigen Nummern/Listen, gegen die ein-/ bzw. ausgehende Nummern geprüft werden. Um eine Nummer hinzuzufügen, geben Sie diese um Textfeld unterhalb der Liste ein und klicken Sie den Knopf Hinzufügen. Um eine bereits bestehende Liste aus einer Unterabteilung hinzuzufügen wählen Sie in der rechten Liste die Unterabteilung, die die Liste enthält und drücken Sie den Knopf Liste zu den Sperrlisten hinzufügen. Um eine oder mehrere Nummern/Listen aus der Liste zu entfernen, markieren Sie diese/n und drücken Sie den Knopf Entfernen.
