Options du compte de messagerie

Gestion des options du compte de messagerie de l'utilisateur :

  • Aucune distribution des messages dans la boite de l'utilisateur : envoie une copie à l'adresse mentionnée dans transférer les messages vers , sans en mettre une copie dans la messagerie de l'utilisateur.
  • Activer la notification d'absence : envoie le message écrit dans message d'absence.
  • Déplacer les messages ayant un niveau de spam supérieur à (a) vers le répertoire (b) : envoie les messages ayant un score supérieur à (a) dans le répertoire (b).
  • Rejeter les messages plus gros que (c) MB : ne pas accepter les messages plus gros que (c) MB.
  • Utilisez les boutons Ajouter ou Ajouter en local pour la gestion des adresses :

    - Le bouton Ajouter en local permet de sélectionner parmis la liste des adresses locales.

    - Le bouton Ajouter permet l'ajout d'adresses extérieures au système.

    - Utilisez le bouton Supprimerpour effacer les adresses de messagerie.
